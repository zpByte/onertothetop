# ONER宇宙生存模拟器 MVP 初版

## 1. 项目定位

暂定名：`ONER宇宙生存模拟器：试运行30天`

类型：文字选择 + 数值生存 + 随机事件 + 捡手机文学剧情

核心体验：玩家进入一个以 ONER 公开演艺生态为灵感的内娱宇宙，在 30 天内处理工作、粉圈、舆情、关系和自我精神状态。玩家不是默认恋爱女主，而是一个被卷入团体生态的人。游戏的爽点来自“活下来”“选得妙”“翻车好笑”“结局有梗”。

设计边界：所有成员相关剧情应基于公开舞台形象、采访风格、团队工作场景和粉圈生态，不写真实私生活推断。正式开发时可使用“角色槽位”或虚构化艺名，避免把现实人物写成不可控的私人关系对象。

## 2. MVP 范围

首版目标：做出可玩的 30 天循环。

首版内容：

- 5 个开局身份
- 6 个核心数值
- 8 个常规行动
- 40 个随机事件
- 6 段捡手机剧情
- 8 个结局
- 1 条事业隐藏线
- 1 条关系隐藏线
- 1 条粉圈翻车线

不做内容：

- 不做完全自由聊天
- 不做复杂地图
- 不做真实社交平台接入
- 不做无限生成剧情
- 不做重度恋爱攻略系统

## 3. 玩家身份

### 3.1 站姐

开局描述：你本来只是想拍几张好图，结果发现自己比部分工作人员还懂行程。

初始数值：

- 心态：70
- 钱包：40
- 事业：20
- 隐蔽：45
- 舆情：15
- 关系：20

身份特性：

- 触发行程类事件概率提高
- 获取情报更容易
- 被扒和被挂风险更高
- 钱包消耗更快

专属事件方向：机场、演唱会、代拍冲突、超话审判、图站出圈。

### 3.2 品牌 PR

开局描述：你只是来对接商务的，结果甲方、乙方、粉丝和艺人团队都开始找你。

初始数值：

- 心态：65
- 钱包：65
- 事业：60
- 隐蔽：55
- 舆情：10
- 关系：15

身份特性：

- 商务类事件收益更高
- 更容易接触团队
- 关系增长较慢
- 决策失误会直接影响事业

专属事件方向：品牌直播、物料泄漏、商务站台、甲方改稿、粉丝维权。

### 3.3 后台助理

开局描述：你负责的工作很小，但你出现的位置都很危险。

初始数值：

- 心态：60
- 钱包：50
- 事业：35
- 隐蔽：70
- 舆情：5
- 关系：25

身份特性：

- 近距离事件概率提高
- 隐蔽基础高
- 心态消耗较快
- 容易触发工作事故

专属事件方向：后台递东西、耳返失误、临时补位、工作人员群聊、深夜收工。

### 3.4 同公司练习生

开局描述：你以为自己在准备舞台，实际上你在准备接受内娱综合素质考试。

初始数值：

- 心态：75
- 钱包：35
- 事业：45
- 隐蔽：50
- 舆情：20
- 关系：20

身份特性：

- 事业线成长最快
- 更容易触发同台和练习室事件
- 容易被粉圈审视
- 舆情增长更敏感

专属事件方向：练习室、同台彩排、师兄师妹争议、公司资源分配、舞台失误。

### 3.5 普通老同学

开局描述：你只是很久没联系的老同学，直到某天一张旧照片开始在网上流传。

初始数值：

- 心态：80
- 钱包：55
- 事业：25
- 隐蔽：35
- 舆情：25
- 关系：35

身份特性：

- 关系基础最高
- 容易触发回忆类剧情
- 隐蔽低
- 一旦被扒，舆情涨幅极高

专属事件方向：旧照、同学聚会、朋友圈截图、同款考古、真假嫂子争议。

## 4. 核心数值

### 4.1 心态

范围：0-100

含义：玩家的精神稳定程度。心态归零会提前进入崩溃类结局。

常见变化：

- 补觉：+8
- 被挂：-15
- 工作成功：+5
- 舆情爆发：-20

### 4.2 钱包

范围：0-100

含义：可支配资源。跑行程、压图、买装备、处理危机都会消耗钱包。

钱包低于 10 时，部分行动不可选。

### 4.3 事业

范围：0-100

含义：玩家自己的发展程度。事业越高，越容易从“路人”进入“合作名单”。

事业达到 80 且舆情低于 50 时，可进入事业隐藏线。

### 4.4 隐蔽

范围：0-100

含义：玩家不被粉圈、营销号、路人扒出的能力。

隐蔽低于 20 时，每周结算会追加一次被扒判定。

### 4.5 舆情

范围：0-100

含义：外部讨论和危险程度。舆情越高，事件越刺激，但也越容易翻车。

舆情高于 70 时，进入危机周。

### 4.6 关系

范围：0-100

含义：玩家与 ONER 宇宙核心人物之间的信任、熟悉和共犯感。关系不是单纯恋爱好感。

关系达到 75 且隐蔽高于 50 时，可进入关系隐藏线。

## 5. 每日行动

每天 3 个行动点。每个行动消耗 1 点。

### 5.1 蹲行程

效果：

- 钱包 -6
- 心态 -4
- 可能触发行程事件
- 站姐身份额外获得情报

### 5.2 做数据

效果：

- 心态 -3
- 事业 +2
- 关系 +1
- 舆情小概率 -3

### 5.3 接工作

效果：

- 心态 -5
- 事业 +6
- 钱包 +8
- 可能触发工作事故

### 5.4 打探消息

效果：

- 心态 -3
- 隐蔽 -3
- 可能解锁隐藏事件

### 5.5 发朋友圈

效果：

- 心态 +4
- 隐蔽 -6
- 舆情小概率 +10
- 老同学身份风险翻倍

### 5.6 装不熟

效果：

- 隐蔽 +6
- 关系 -2
- 舆情 -3

### 5.7 补觉

效果：

- 心态 +10
- 钱包 +0
- 错过一个随机事件判定

### 5.8 处理舆情

效果：

- 钱包 -8
- 心态 -5
- 舆情 -10
- 隐蔽 +2

## 6. 周结算

每 7 天触发一次周结算。

结算内容：

- 如果心态低于 20，触发崩溃预警
- 如果隐蔽低于 20，触发被扒判定
- 如果舆情高于 70，下一周进入危机周
- 如果事业高于 60，增加工作机会事件权重
- 如果关系高于 60，增加捡手机剧情权重

危机周规则：

- 每天额外触发 1 个舆情事件
- 所有公开行动的舆情涨幅 +50%
- 处理舆情行动效果提升，但钱包消耗增加

## 8. 首批事件样例

### 8.1 机场同框三秒

文本：你刚走出通道，身后突然有人喊了一声。你下意识回头，结果正好和他们同框。五分钟后，超话出现一张高糊图。

选项：

- 立刻换头像装死：隐蔽 +5，心态 -3
- 发朋友圈“今天只是路过”：舆情 +8，关系 -2
- 找熟人压图：钱包 -10，隐蔽 +10
- 反手发一张更糊的：舆情 +15，触发捡手机剧情

### 8.2 物料提前泄漏

文本：品牌方群里突然多出一张还没公开的宣传图。十秒后，有人撤回了。但你已经看见了。

选项：

- 假装没看见：隐蔽 +4，事业 +2
- 提醒负责人：事业 +8，关系 +2，心态 -4
- 截图保存：隐蔽 -8，触发风险判定
- 发给朋友吐槽：舆情 +12，心态 +3

### 8.3 工作群误发

文本：你把一句“这版妆造真的很难评”发进了 47 人工作群。

选项：

- 立刻撤回：心态 -8，隐蔽 +2
- 解释是发错群：事业 -4，心态 -4
- 装作在提优化建议：事业 +5，关系 -2
- 继续补充三条专业意见：事业 +10，舆情 +5

### 8.4 深夜消息

文本：凌晨 1:17，你收到一句：“今天辛苦了，早点睡。”你盯着屏幕，觉得这比 KPI 还难处理。

选项：

- 秒回：关系 +6，隐蔽 -3
- 五分钟后回：关系 +4
- 第二天早上回：隐蔽 +4，关系 -1
- 不回但截图：心态 +5，触发捡手机剧情

### 8.5 超话开庭

文本：有人把你的旧账号翻出来了。评论区已经开始从头像分析到 IP 属地。

选项：

- 立刻注销账号：隐蔽 +8，心态 -10
- 发长文解释：舆情 +20，事业 -5
- 找朋友帮忙控评：钱包 -12，舆情 -8
- 什么都不做：心态 -5，进入随机判定

### 8.6 彩排临时补位

文本：有个工作人员突然问你：“你会不会走位？先上去帮忙站一下。”

选项：

- 直接上：事业 +8，关系 +3，心态 -4
- 拒绝：隐蔽 +4，事业 -2
- 问清楚再上：事业 +5，心态 -2
- 上去后走错位：舆情 +8，触发事故判定

## 9. 捡手机文学模块

定位：捡手机剧情是主线事件后的高情绪片段，用来制造截图传播点。它不是自由聊天系统，而是带分支的剧情 UI。

形式：

- 仿聊天 App 私聊
- 仿工作群
- 仿微博时间线
- 仿 Twitter 时间线
- 仿备忘录
- 仿热搜榜

每段捡手机剧情包含：

- 剧情入口条件
- 固定聊天记录
- 2-4 个玩家回复选项
- 每个选项影响数值
- 可选隐藏标记

### 9.1 捡手机数据格式

```json
{
  "id": "phone_story_midnight_001",
  "type": "chat",
  "title": "凌晨1:17",
  "messages": [
    { "sender": "对方", "text": "今天辛苦了，早点睡。", "time": "01:17" },
    { "sender": "玩家", "text": "输入中...", "time": "01:18", "pending": true }
  ],
  "choices": [
    {
      "label": "你也是，今天舞台很好。",
      "reply": "你也是，今天舞台很好。",
      "effects": { "关系": 5, "心态": 2 }
    },
    {
      "label": "收到，老板。",
      "reply": "收到，老板。",
      "effects": { "隐蔽": 3, "关系": -1 }
    },
    {
      "label": "你怎么还没睡？",
      "reply": "你怎么还没睡？",
      "effects": { "关系": 7, "隐蔽": -4 }
    }
  ]
}
```

### 9.2 首批捡手机剧情

#### A. 凌晨 1:17

入口：触发“深夜消息”事件。

聊天：

- 对方：今天辛苦了，早点睡。
- 玩家：输入中...

选项：

- 你也是，今天舞台很好。：关系 +5，心态 +2
- 收到，老板。：隐蔽 +3，关系 -1
- 你怎么还没睡？：关系 +7，隐蔽 -4

#### B. 工作群误发后续

入口：触发“工作群误发”事件，并选择“继续补充三条专业意见”。

聊天：

- 项目负责人：你刚刚说的第三点，展开讲讲。
- 同事 A：我也觉得有道理。
- 同事 B：她不会真是来上班的吧？

选项：

- 继续专业分析：事业 +10，心态 -6
- 说自己只是随口一提：隐蔽 +5，事业 -3
- 私聊负责人发完整方案：事业 +12，关系 +2，钱包 +5

#### C. 微博时间线：高糊图之夜

入口：机场同框事件中选择“反手发一张更糊的”。

时间线：

- 营销号：某团成员机场身边神秘女生引热议。
- 粉丝甲：这图糊得我家 WiFi 都不认。
- 粉丝乙：别什么都嫂，能不能先看清是人还是柱子。
- 路人：内娱每天都在考验我的视力。

选项：

- 点赞“柱子”评论：心态 +6，隐蔽 -5
- 举报营销号：舆情 -4，心态 -2
- 切小号加入讨论：心态 +4，舆情 +8

#### D. 旧照片被翻出

入口：普通老同学身份，隐蔽低于 40。

聊天：

- 老同学 A：这是不是你？
- 老同学 B：我靠这张照片怎么被翻出来了？
- 玩家：？
- 老同学 A：你现在说你俩不熟还来得及吗？

选项：

- 坚持不熟：隐蔽 +6，关系 -3
- 说只是同学：舆情 +6，心态 -3
- 让群友闭嘴：隐蔽 +8，钱包 -5

#### E. 品牌直播前一小时

入口：品牌 PR 身份，事业高于 50。

聊天：

- 甲方：能不能临时加一个互动问题？
- 经纪团队：问题先发我们审核。
- 主持人：我已经在化妆间了，谁给我最终版？

选项：

- 坚持走审核：事业 +6，关系 +2
- 直接给主持人：事业 +3，舆情 +8
- 删掉互动环节：事业 -4，隐蔽 +5

#### F. 练习室门口

入口：同公司练习生身份，事业高于 45。

聊天：

- 队友：你刚才是不是被叫去隔壁练习室了？
- 玩家：嗯，帮忙看了下走位。
- 队友：你小心点，论坛已经有人在猜了。

选项：

- 继续练自己的：事业 +5，隐蔽 +2
- 去论坛看看：心态 -8，舆情 +4
- 找工作人员确认安排：事业 +4，关系 +2

## 10. 结局设计

结局在第 30 天结算时触发。优先级从特殊失败、隐藏线到普通结局。

### 10.1 团没塌，你先塌了

条件：心态 <= 0。

结局文案：你看完最后一条评论，突然觉得整个内娱都不值得。第二天，你把所有 App 移到文件夹最后一页，命名为“少管我”。

### 10.2 被扒得很完整，但嘴硬得很成功

条件：隐蔽 <= 10，舆情 >= 70，心态 > 0。

结局文案：你的旧账号、旧头像、旧朋友圈和小学作文都被翻了出来。你唯一的胜利是：没有一句承认。

### 10.3 事业线通关：合作名单常驻

条件：事业 >= 80，舆情 < 50。

结局文案：一开始没人知道你是谁。三十天后，所有项目表格里都多了一行你的名字。

### 10.4 关系线通关：没有公开，但所有人都懂了

条件：关系 >= 75，隐蔽 >= 50，舆情 < 60。

结局文案：你们没有多说什么。只是后来每一次混乱现场，总有人下意识先看向你。

### 10.5 超话黑名单永久居民

条件：舆情 >= 85，隐蔽 < 40。

结局文案：你的名字成了超话违禁词。某种意义上，你也算拥有了专属词条。

### 10.6 低调路人，实则隐藏大佬

条件：隐蔽 >= 80，事业 >= 50。

结局文案：没人拍到你，没人扒到你，但每次关键节点都有你的痕迹。粉圈称之为玄学，项目组称之为靠谱。

### 10.7 你没有成为嫂子，你成为了项目负责人

条件：事业 >= 70，关系 < 60。

结局文案：故事没有走向暧昧。故事走向了排期、预算、复盘会和所有人都找你要最终版。

### 10.8 试运行存活

条件：没有触发以上结局。

结局文案：三十天过去，你没有大红，也没有大翻车。你只是更清楚地知道，内娱真正的怪不是别人，是每周一的待办。

## 12. 文案基调

关键词：

- 荒诞
- 嘴硬
- 高压
- 缺德但不恶毒
- 像热评，但更有剧情
- 像打工，但每一步都能上热搜

文案原则：

- 每个事件文本控制在 60-120 字
- 每个选项尽量短，适合截图
- 不靠露骨恋爱制造爽点
- 让玩家觉得“这事离谱，但内娱好像真干得出来”
- 成员相关剧情保持模糊空间，重点写场景压力和玩家选择
